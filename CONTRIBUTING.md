# Contributing

Thanks for contributing!

## Development

Requirements:
- Bun
- `codex` installed and available in `PATH` (only needed for manual end-to-end testing)

Commands:
- Install deps: `bun install`
- Typecheck: `bun run typecheck`
- Tests: `bun test`
- Build (Node CLI): `bun run build`

## Pull requests

- Keep changes focused and easy to review.
- Add/update tests when behavior changes.
- Update documentation (`README.md` / `docs/how-it-works.md`) when UX changes.
